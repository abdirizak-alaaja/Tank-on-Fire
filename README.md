# Tank on Fire 🔥

**Tank on Fire** is an explosive, top-down 2D arcade shooter built entirely in Python using the **Pygame** library. Jump into intense arena combat, navigate destructible environments, and outmaneuver enemy tanks to survive!

## Features
- **Dynamic Pre-Game Menu:** Seamlessly select between different interactive environments (Grass, Sand, Dirt) using keyboard or mouse.
- **Rich Assets & Graphics:** Features beautifully scaled tank sprites, accurate rotating barrels, colored enemy variations, and custom projectiles.
- **Interactive Environments:** The battlefield is natively layered with destructible and solid obstacles like sandbags and explosive barrels that require tactical navigation!
- **Fluid Combat:** Fast-paced Arrow Key movement synced with multi-directional firing physics and precise hitbox tracking!
- **Custom Visual Effects:** Tanks detonating organically trigger dynamic colored smoke particle animations and immersive explosion SFX.

## How to Play
1. **Launch:** Run `python main.py`
2. **Select World:** Use `LEFT`/`RIGHT` keys (or your mouse cursor!) to highlight your favorite map, and press `ENTER` to deploy!
3. **Controls:**
   - **`UP` `DOWN` `LEFT` `RIGHT`**: Drive your Tank!
   - **`SPACE`**: Fire the primary cannon!
4. **Objective:** Dodge enemy fire, blast through destructible barrels, and annihilate the randomly generating enemy tanks to survive!

## Requirements
- Python 3.8+
- Pygame (`pip install pygame`)
